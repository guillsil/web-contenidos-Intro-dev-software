Supongamos que tienes una función simple llamada suma que toma dos números y devuelve su suma:

def suma(a, b):
return a + b

En este ejemplo, hemos definido tres pruebas en la clase TestSuma. Cada método dentro de esta clase comienza con el prefijo test_, lo que indica a unittest que debe ejecutarlo como una prueba. Usamos métodos de aserción como assertEqual para verificar si la función suma produce los resultados esperados para diferentes casos.

Para ejecutar estas pruebas, simplemente ejecuta el script Python. Si todas las pruebas pasan, no verás ninguna salida. Si alguna prueba falla, verás un mensaje indicando qué prueba falló y cuál era el resultado esperado.
