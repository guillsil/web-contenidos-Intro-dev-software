Modificación de Link:

Toda referencia a links dentro del sitio o cualquier redirección se deberá crear mediante:

url_for('user', name='john', _external=True)
  • _external=True - Crea la dirección absoluta, ideal cuando necesitamos enviar un link por correo electrónico, por ejemplo.

Modificación de Archivos Estáticos:

Toda referencia a recursos estáticos como archivos javascript, imágenes, CSS, etc. se deberá crear mediante:

url_for('static', filename='css/styles.css', _external=True)
  • static - Hará referencia a la carpeta "static".
  • filename - Informará la carpeta y el archivo particular a acceder.
