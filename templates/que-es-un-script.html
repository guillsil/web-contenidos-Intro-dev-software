¿Qué es un script?

Un script es una secuencia de comandos y operaciones que el shell puede interpretar y ejecutar.

Guardando un script

Esta secuencia de comandos se suelen guardar en un archivo de texto con la extensión .sh (por ejemplo, mi_script.sh) que luego se podrá ejecutar utilizando el comando bash, ./, sh, etc., según el shell que estemos utilizando.

Automatización de tareas

De esta forma, podremos automatizar tareas.

Características de un script

  • Ejecutarse sin errores
  • Realizar la tarea para la que está destinado
  • Tener la lógica del programa claramente definida
  • Evitar el código innecesario
  • Ser reutilizable
