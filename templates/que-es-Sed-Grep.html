El comando grep

Sirve para buscar palabras en un archivo, así como patrones de palabras en archivos. Se usa de la siguiente manera:

  • Buscar una palabra: grep "Hola" ejemplo.txt
  • Buscar una frase en un archivo: grep "Hola mundo" ejemplo.txt
  • Buscar múltiples palabras: grep búsqueda1 archivo | grep búsqueda2 archivo
  • Buscar una palabra en un conjunto de archivos: grep -l palabra_a_buscar ./*

El parámetro -c se utiliza para contar.

El comando sed

Se utiliza para reemplazar palabras de un archivo. Se usa de la siguiente manera:

  • sed -i 's/old-text/new-text/g' input.txt

Donde:

  • -i: Indica hacer un update en el archivo
  • s: Indica sustituir las palabras
  • g: Indica hacerlo en forma global
