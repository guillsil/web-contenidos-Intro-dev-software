Bases de Datos (BBDD) ó Data Bases (DB):

¿Qué es?

  • Herramienta para recopilar y organizar información.
  • Conecta datos en una unidad lógica.
  • Nos resuelve la problemática de manejar datos.

Las hay de tipo

  • Relacionales
  • No Relacionales

Bases de Datos Relacionales → ¡SQL!

  • Almacenan los datos en tablas con columnas y filas.
  • Cada columna representa un atributo de datos específico y cada fila representa una instancia de esos datos.

Bases de Datos No Relacionales

Existen varios sistemas de bases de datos no relacionales diferentes debido a las variaciones en la forma en que administran y almacenan los datos sin esquemas. Los datos sin esquema son datos que se almacenan sin las restricciones que requieren las bases de datos relacionales.

Ejemplo:

  • mongoDB
  • neo4j

Claves:

  • A cada tabla se le asigna una clave principal, una columna de identificador que identifica de forma exclusiva la tabla.
  • La clave principal se utiliza para establecer relaciones entre las tablas.
  • Una vez que se conectan dos tablas, se obtienen datos de ambas con una sola consulta.
