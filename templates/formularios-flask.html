Creación de Formularios:

Los formularios dentro de un sitio web son fundamentales para poder interactuar con el usuario. Si bien Flask posee una dependencia que agiliza y permite crear formularios más robustos y seguros, en la materia vamos a utilizar los métodos tradicionales de etiquetas <form> y métodos POST para enviar la información.

Definición de un formulario en HTML:

Definición de form-handler (lado servidor)

Debemos definir el tipo de método que recibirá la ruta, en este caso GET (por defecto) + POST (para el envío de la información)

Si recibimos información con el método POST, entonces redirigimos a un sitio de respuesta, sino volvemos a mostrar el formulario.
